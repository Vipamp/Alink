## 1、pyalink 环境搭建
```
pip install pyalink-flink-1.9
pip install pyalink
```
> 备注：
> 1.pyalink 和 pyalink-flink-*** 不能同时安装，如果之前安装过，先卸载掉：pip uninstall xxx
> 2.pyalink 或者 pyflink 都需要安装jdk。
